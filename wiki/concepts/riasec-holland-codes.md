# RIASEC / Holland Codes

**Also known as:** Holland Codes, Holland Occupational Themes  
**Origin:** Developed by psychologist John L. Holland  
**Type:** Career interest psychometric framework

## Six types
| Code | Type | Description |
|------|------|-------------|
| R | Realistic | Hands-on, mechanical, outdoor work |
| I | Investigative | Analytical, scientific, research-oriented |
| A | Artistic | Creative, expressive, unstructured work |
| S | Social | Helping, teaching, collaborative work |
| E | Enterprising | Leading, persuading, business-oriented |
| C | Conventional | Organised, data-driven, rule-following |

Individuals are typically characterised by a 2–3 letter combination (e.g., "ASE").

## Use in Schola
[[schola]]'s **Career Quiz** (Feature 1) is built on RIASEC. The quiz asks preference questions ("What makes you happy?") and maps answers to a Holland type. Output is a **career personality card** with a local, relatable label (e.g., "Berdarah Seni" for artistic types) and a list of matched jobs.

This approach grounds Schola's recommendations in an empirically validated framework while presenting it in an accessible, visual format for Malaysian secondary-school students.
